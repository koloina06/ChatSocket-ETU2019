# ChatSocket-ETU2019

Fonctionnalité chat socket:
-Compilena ny server.bat
-Ny server no mampiseo ny liste clients connectes
-Compilena ny clients
-Ny client ray mila mapiditra anarana sy le host hidirany amzay connecté amle chat groupé
-Rehefa mi-cliquer bouton send ny client dia tafiditra ao amin'ilay frame misy an'ilay liste clients connectés ary lasa any am clients rehetra ny message
-Clients sy clients no mifampiresaka
-Afaka misy clients maro connectes amin'ny alalan'ilay multi-thread
